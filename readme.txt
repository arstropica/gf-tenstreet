This plugin adds web service functionality to Gravity Forms used to submit applications to the TenStreet API.

== Description ==
This plugin adds web service functionality to Gravity Forms used to submit applications to the TenStreet SOAP web service.

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress


== Frequently Asked Questions ==
= Can I use the plugin as-is? =

Yes. You will need to enter access credentials from TenStreet to submit to their API.
